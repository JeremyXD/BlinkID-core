Detailed Description

Settings for configuring Machine Readable Travel Document recognizer. MRTD recognizer can detect machine readable zone in the image and extract data from it. Besides that, it can also perform recognition of non-MRZ data on known locations relative to document location. For more information about how to use that, check Android documentation - the only difference is that MRTDDocumentClassifier in C API is actually a ClassifierFunction defined in TemplatingSettings.h To learn how to use Templating API to also scan non-MRZ data on example, check this article and also check demo apps provided with the SDK.

◆ templatingSettings

TemplatingSettings* MRTDSettings::templatingSettings

Pointer to additional templating settings that can be used to perform recognition of non-MRZ data. If you are only interested in MRZ data, set this to NULL, othervise use mrtdSettingsInitialize to initialize it. When done, you must terminate it with mrtdSettingsTerminate.
