 Transformation matrix of the OCR result. All coordinates of OcrBlock, OcrLine and OcrChar are given in coordinate system of the image the OCR was performed on. However, in some cases this is not the same image that was given as input. Detailed Description

Structure that describes OCR recognition result. OCR recognition result is a list of blocks (see OcrBlock), each block having a list of lines (see OcrLine) and each line having a list of characters (see OcrChar). Each OcrChar contains all information that OCR engine has returned. OcrResult structure has its own lifetime, i.e. it can outlive RecognizerResult object it originated from. To delete the OcrResult structure and cleanup the memory, you should use function ocrResultDelete.
